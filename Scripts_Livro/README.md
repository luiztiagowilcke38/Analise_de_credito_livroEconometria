# Scripts de Econometria Avançada: R & Python

Este repositório contém uma coleção de scripts de alta performance desenvolvidos em **R** e **Python**, focados em resolver problemas complexos de econometria financeira e social.

## Autor
**Luiz Tiago Wilcke**

## Origem das Ideias
Os códigos aqui presentes foram extraídos, adaptados e expandidos a partir do manuscrito original do livro:
**"Métodos Econométricos: Teoria e Prática"** (Trabalho em andamento), autoria de Luiz Tiago Wilcke.

### Detalhamento por Capítulo:

1.  **Capítulo 19: Finanças Quantitativas: Volatilidade Condicional (GARCH), Cópulas e Integração Fracionária**
    *   **Modelo de Saltos de Merton (Jump-Diffusion):** Implementado para capturar descontinuidades súbitas em séries temporais (crashes financeiros e crises sociais).
    *   **DCC-GARCH (Dynamic Conditional Correlation):** Usado para analisar o contágio entre o mercado brasileiro (IBOVESPA) e o internacional (S&P 500).
    *   **GARCH-EVT (Extreme Value Theory):** Modelagem de caudas pesadas para cálculo de VaR e Expected Shortfall (ES).

2.  **Capítulo 20: Econometria Bayesiana Aplicada e Inferência Paramétrica Estocástica**
    *   **BVAR (Bayesian VAR) com Minnesota Prior:** Previsão macroeconômica robusta para PIB, Inflação e Juros sob incerteza.
    *   **Inferência Bayesiana via MCMC (NUTS):** Regressão avançada utilizando o amostrador Hamiltoniano para modelagem de indicadores sociais.

## Aplicações Práticas
-   **Mercado Financeiro:** Gestão de risco extremo, arbitragem estatística e análise de volatilidade.
-   **Sociedade:** Projeções macroeconômicas, análise de contágio de crises e modelagem de incerteza em políticas públicas.
